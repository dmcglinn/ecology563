Assignment 11

Due Date

Friday, November 30, 2012

Data Source

The data set for this assignment is quinn1.csv, the comma-delimited text file that you analyzed in Assignment 1.

Questions

  1. Refit your final egg count rate model from Assignment 1 as a Bayesian model. This should be a model with a normally distributed response in which there is a two-factor interaction between Density and Season.
    1. Provide diagnostics that indicate that the Markov chains have converged to the corresponding posterior distributions and are properly mixing.
    2. Be sure to submit your BUGS program code and the R code you used to construct the variables and objects that you passed to WinBUGS or JAGS.
  2. Obtain 95% percentile and 95% HPD credible intervals for all the treatment effects.
  3. Graph the estimated posterior distributions for all the parameters in the model.
  4. Obtain estimates of the six treatment means as well as 95% percentile credible intervals for these means. Compare these values to what you obtained in Assignment 2.
